# 笔记分类功能使用指南

## ✅ 实现完成

当前版本为「手动标签 + 支持创建自定义标签」，已集成到创建笔记流程中。

## 🚀 如何使用

### 1. 首次启动
应用会自动创建 6 个预设标签（Tag）：
- 🧳 **Work**
- 🏠 **Life**
- 📚 **Study**
- 💡 **Ideas**
- ✅ **Todo**
- 📁 **Other**

### 2. 创建带分类的笔记

#### 语音笔记
1. 点击黄色录音按钮
2. 说出笔记内容，例如："明天下午3点开会讨论新项目"
3. 点击确认结束录音
4. 完成转录后，标签选择器弹出
5. 你可以：
   - 选择一个或多个标签，然后点击 **"Confirm"**
   - 点击右上角 **"New Tag"** 创建自定义标签（创建后可直接选中）
   - 点击 **"Skip"** 跳过打标签（仍会保存笔记）
6. 笔记保存完成，带有选定的分类标签

#### 文字笔记
1. 在底部输入框输入笔记内容
2. 点击发送
3. 标签选择器弹出（与语音笔记一致）
4. 选择标签后 **"Confirm"** 保存，或 **"Skip"** 直接保存

### 3. 筛选笔记

在首页 "Recent Notes" 标题下方，你会看到分类筛选栏：

```
🏷️ [全部] [工作] [生活] [学习] [想法] [待办] [其他]
```

- **点击任意分类**：只显示该分类的笔记
- **再次点击**：返回"全部"视图
- **左右滑动**：查看更多分类
- **数字显示**：每个分类显示包含的笔记数量

### 4. 查看笔记分类

在笔记卡片上，你会看到彩色的分类标签：

```
┌─────────────────────────────┐
│ 明天下午3点开会讨论新项目...  │
│                             │
│ 🧳 工作                      │  ← 分类标签
│                             │
│ 2026-01-13 20:30 · 2小时前   │
└─────────────────────────────┘
```

## 🎨 设计亮点

### 自定义标签
- 随时创建新标签，不受预设数量限制
- 颜色与图标可选，便于识别和筛选
- 一条笔记可添加多个标签

### 流畅动画
- 分类切换使用 Spring 动画
- 选中状态有缩放效果
- 列表筛选带淡入淡出动画

### 视觉清晰
- 每个分类有独特的颜色和图标
- 选中状态使用填充背景
- 未选中状态使用边框和浅色背景

## 📊 示例场景

### 场景 1：工作会议
**录音内容**：
> "明天下午3点和产品团队开会，讨论Q2的新功能规划，记得准备PPT"

**结果**：你可以选择 **Work**，也可以额外创建/选择 **Meeting** 等自定义标签

### 场景 2：生活琐事
**录音内容**：
> "周末去超市买牛奶、鸡蛋、面包，顺便取快递"

**结果**：你可以选择 **Life** + **Todo**

### 场景 3：学习笔记
**录音内容**：
> "今天学习了 SwiftUI 的动画系统，Spring 动画的 dampingFraction 参数控制弹性"

**结果**：你可以选择 **Study**

## 🔧 技术细节

### 数据迁移
- 如果你已有旧数据，首次启动时会自动创建分类
- 旧笔记不会自动添加分类，但可以手动添加

### 性能优化
- 分类筛选使用内存过滤，速度极快
- 支持数千条笔记的流畅筛选

### 同步支持
- 分类信息会同步到服务器
- 多设备间分类保持一致

## 🐛 已知限制

1. **暂不支持标签管理**
   - 目前支持创建并使用自定义标签
   - 重命名/删除/排序等管理能力可在后续加入（建议做一个 Tag 管理页）

3. **不支持批量编辑**
   - 暂时无法批量修改笔记的分类
   - 批量操作计划在 Phase 2 实现

## 💡 使用建议

1. **多标签更灵活**：一条笔记可以有多个标签，比如 **Work + Todo**
2. **按需创建**：当预设不够用时，直接在选择器里点 **"New Tag"**
3. **定期筛选**：使用标签筛选快速找到特定类型的笔记
4. **跳过也可以**：不是所有笔记都需要标签，点击 **"Skip"** 直接保存
