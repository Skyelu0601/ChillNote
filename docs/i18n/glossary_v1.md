# ChillNote Glossary v1

## Brand Terms
- ChillNote: 品牌名，不翻译
- Chill AI: 功能品牌名，不翻译
- PRO: 订阅等级标签，可保留大写

## Product Terms
- Note: 笔记
- Tag: 标签
- Recycle Bin: 回收站
- Voice Transcription: 语音转写
- Subscription: 订阅
- Export: 导出
- Pending Recordings: 待处理录音

## Style Rules
- 中文（简/繁）：使用自然口语，不直译英文语序。
- 日语：按钮尽量简短，避免英文夹杂。
- 法/德/西/韩：保持操作文案统一敬语和语气。
- 错误提示：先说问题，再给动作建议。
